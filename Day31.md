#  AI Supply Chain Control Tower

An interactive operations management simulator that places the user in the role of **Head of Operations** for a global supply chain organization. Built as a single-file HTML application using **HTML, CSS, and Vanilla JavaScript**, the simulator challenges users to monitor live operational alerts, prioritize incidents, make strategic decisions, and optimize business performance before time runs out.

---

##  Overview

AI Supply Chain Control Tower simulates a real-world logistics control center where operational disruptions occur continuously. Players must analyze incoming alerts, select the most appropriate corrective actions, and balance key business metrics such as service level, inventory health, transportation efficiency, operating cost, and customer satisfaction.

The project is designed as an educational simulation that demonstrates how operational decisions directly influence supply chain performance.

---

##  Features

-  Real-time Operations Dashboard
-  Randomly Generated Supply Chain Alerts
-  Live Countdown Timer
-  Dynamic KPI Updates
-  Score-Based Decision System
-  Increasing Difficulty Throughout Gameplay
-  Live Event Log
-  Multiple Corrective Actions per Incident
-  Priority-Based Alert System
-  Responsive User Interface
-  Pause Functionality
-  Sound Toggle (Visual)
-  Help & Instructions Modal
-  Final Performance Dashboard
-  Replay Simulation

---

##  Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript

> No frameworks, backend, APIs, or external libraries are required. The application runs completely offline.

---

##  Simulation Workflow

1. Launch the application
2. Read the simulation instructions
3. Monitor incoming operational alerts
4. Analyze alert priority and business impact
5. Select the most effective corrective action
6. Observe KPI and score changes in real time
7. Manage multiple simultaneous incidents
8. Complete the simulation before the timer expires
9. Review the final operational performance dashboard
10. Replay with a new set of randomized scenarios

##  Key Performance Indicators (KPIs)

The simulator tracks the following operational metrics:

- Service Level
- Customer Satisfaction
- Inventory Health
- Transportation Efficiency
- Operating Cost
- Revenue Protected
- Overall Score
- Remaining Time

---

##  Operational Alert Types

The simulator includes randomly generated scenarios such as:

- Port Congestion
- Supplier Delay
- Truck Breakdown
- Warehouse Stockout
- Customs Inspection
- Demand Spike
- Factory Machine Failure
- Weather Disruption
- Inventory Count Mismatch
- Damaged Shipment

Each alert includes:

- Priority Level
- Business Impact
- Time Remaining
- Recommended Response Options
