# Day 48 – The Verdict Engine 

##  Project Overview

**The Verdict Engine** is a single-file interactive HTML application that helps users make confident decisions through data-driven comparisons. It researches and compares multiple options based on user-defined criteria, provides transparent citations, and dynamically updates rankings as users adjust the importance of each criterion.

The application follows a guided interview process, researches real-world data, and presents a professional decision dashboard with complete transparency into the research process.

---

##  Features

- Guided MCQ-based interview flow
- Compare multiple real-world options
- Adjustable criteria weighting
- Live ranking updates
- Research-backed data with citations
- Sources panel for every data point
- "How This Was Researched" documentation
- Loading and empty state handling
- Responsive modern UI
- Offline-ready single HTML file

---
