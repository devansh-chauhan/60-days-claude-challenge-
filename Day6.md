# Day 6 – ATS Resume Optimization with Claude

## Objective

Learn how AI can analyze and optimize a resume for Applicant Tracking Systems (ATS) and recruiter readability.

---

## Original Resume

I uploaded my existing resume and asked Claude to optimize it using the provided ATS Resume Optimizer prompt.

### Resume Highlights

- B.Tech Computer Science & Engineering (AI & ML)
- Python, C, C++
- HTML, CSS, JavaScript, React.js, Node.js
- Personal Portfolio Website
- Eco-Learn Platform
- Reminder Bot
- AI & Python Certifications
- Smart India Hackathon Participation

---

## Prompt Used

You are an ATS optimization expert and resume writer.

Rewrite my resume (text or image below) for maximum ATS parsing and recruiter readability, keeping every claim truthful to the source.

(Challenge Prompt)

---

## ATS Analysis Generated by Claude

### ATS Score

- Previous Score: 42/100
- Optimized Score: 84/100

### Key Improvements Suggested

1. Added a stronger Professional Summary.
2. Improved keyword alignment for AI/ML and Software Development roles.
3. Reorganized sections into ATS-friendly order.
4. Improved project descriptions using stronger action verbs.
5. Reduced redundancy and improved readability.
6. Used a clean single-column ATS-friendly layout.
7. Better skill categorization.
8. Improved recruiter scanning experience.

---

## Optimized Resume Highlights

### Professional Summary

Added a concise summary highlighting:

- AI & ML specialization
- Python development
- Full-stack web development
- Competitive programming experience

### Improved Project Presentation

Projects were rewritten using stronger action verbs and clearer impact statements.

Examples:

- Personal Portfolio Website
- Eco-Learn Platform
- Reminder Bot

### Skills Organization

Skills were grouped into:

- Programming Languages
- Frontend
- Backend
- Tools & Version Control
- Other Technologies

---

## Comparison

| Aspect | Original Resume | Optimized Resume |
|----------|----------|----------|
| ATS Score | 42/100 | 84/100 |
| Readability | Medium | High |
| Keyword Optimization | Low | High |
| Structure | Basic | ATS-Friendly |
| Professional Summary | Missing | Added |
| Recruiter Scanability | Medium | High |

---

## Biggest Insight

I learned that a resume is not just about listing skills and projects.

The way information is structured, keyword placement, section order, and action verbs significantly impact ATS performance and recruiter perception.

Even without adding new skills or achievements, the ATS score improved dramatically through better presentation.

---

## Key Learnings

1. ATS systems prioritize structure and keywords.
2. Professional summaries help recruiters quickly understand your profile.
3. Strong action verbs make projects more impactful.
4. Clean formatting improves ATS parsing.
5. Resume optimization is about presentation, not exaggeration.
