# Day 49 – Personal AI Playbook

##  Project Overview

**Personal AI Playbook** is a single-file interactive HTML application that helps users build a personalized library of reusable AI workflows. Instead of storing isolated prompts, the application focuses on creating modular AI systems that can be customized, reused, and expanded for different tasks.

The application is fully personalized through an interview process, generating workflows tailored to the user's profession, AI use cases, productivity bottlenecks, and preferred AI models.

---

##  Features

- Interactive onboarding interview
- Personalized AI workflow library
- Modular Prompt Builder
- Autonomous Loop Builder
- Reusable workflow templates
- Editable prompt blocks
- Workflow categories
- Favorite workflows
- Search & filter functionality
- Import & Export workflow library (JSON)
- Local Storage support
- Responsive commercial SaaS UI
- Dark mode
- Keyboard shortcuts
- Smooth animations
- Offline support
