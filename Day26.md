# Day 26 – Prior Authorization Workflow Simulator

##  Project Overview

This project was completed as part of **Day 26 of the 60 Days of Claude Challenge**.

The objective was to build a **Prior Authorization (PA) Workflow Simulator** as a single-page HTML application that visually demonstrates how healthcare prior authorization requests move through different stages before receiving an insurance decision.

The simulator transforms a complex healthcare administrative process into an interactive, educational, and gamified experience using drag-and-drop interactions and workflow visualizations.

---

##  Objectives

- Simulate the complete Prior Authorization workflow
- Educate users about healthcare insurance authorization processes
- Build a responsive application using only HTML, CSS, and Vanilla JavaScript
- Implement interactive workflow progression without any backend
- Provide real-time feedback, educational insights, and workflow summaries

---

##  Features

###  Workflow Simulation

- Patient, Provider, and Payer workflow lanes
- Interactive drag-and-drop case movement
- Multiple healthcare scenarios
- Medical necessity evaluation
- Prior Authorization document collection
- Request submission to payer
- Clinical review process
- Approval workflow
- Pend workflow
- Denial workflow
- Appeal process
- Peer-to-Peer Review simulation

###  Dashboard & Analytics

- Progress tracker
- Journey timeline
- Days elapsed counter
- Efficiency score
- Workflow completion summary
- Educational explanations after every workflow stage

###  Interactive Experience

- Drag-and-drop interactions
- Animated workflow progression
- Celebration animation on approval
- Restart workflow
- New patient scenario generation
- Fully responsive interface

---

##  Patient Scenarios Included

- Elective Surgery
- MRI Authorization
- Specialty Medication
- Inpatient Admission

Each scenario follows a complete Prior Authorization lifecycle with different workflow outcomes.

##  What I Learned

This project helped me understand how Prior Authorization works in the U.S. healthcare system and how technology can simplify complex administrative workflows.

Some key takeaways include:

- Understanding the complete PA lifecycle
- Importance of medical necessity evaluation
- Role of documentation in insurance approvals
- Workflow visualization using frontend technologies
- Managing application state with JavaScript
- Building educational simulations without external dependencies
- Designing user-friendly healthcare interfaces
