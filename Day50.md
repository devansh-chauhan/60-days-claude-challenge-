# Day 50 – Defend Your Experience

## 📌 Project Overview
Today I built **Defend Your Experience**, an AI-powered interview preparation application designed to help users confidently defend every claim they make about themselves.

Unlike traditional interview practice tools, this application analyzes resumes, portfolios, LinkedIn profiles, research papers, startup stories, and other professional documents to generate personalized follow-up questions. It acts like an intelligent interviewer that continuously challenges users to provide stronger evidence, clearer explanations, and more compelling stories.

---

## 🚀 Features

- Interactive interview setup with guided questions
- Personalized UI based on user-selected visual style
- Drag-and-drop upload support
- AI-powered claim extraction from uploaded documents
- Adaptive follow-up questioning based on previous responses
- Intelligent skepticism for deeper interview preparation
- Confidence tracking and progress indicators
- Defense Report highlighting strengths and improvement areas
- Session history using Local Storage
- Export functionality
- Responsive modern interface
- Offline-friendly single HTML application
