#  Day 39 – PDF Splitter & Merger

##  Project Overview

**PDF Splitter & Merger** is a premium single-page web application that enables users to split and merge PDF files directly in the browser. Built with HTML, CSS, and JavaScript, it provides a modern user experience with drag-and-drop support, live previews, page management, and responsive design.

All PDF processing is performed client-side, ensuring privacy and eliminating the need for backend services.

---

##  Features

###  PDF Splitter
- Upload PDF documents
- Automatically detect total pages
- Visual page thumbnails and preview
- Split by:
  - Custom page ranges
  - Individual pages
  - Every N pages
  - Split after selected pages
- Extract multiple page ranges
- Input validation with error highlighting
- Preview output before processing

###  PDF Merger
- Upload multiple PDF files
- Drag-and-drop file upload
- Reorder files before merging
- View page counts and file previews
- Display total files and pages
- Generate and download merged PDF

###  User Experience
- Responsive modern interface
- Dark mode support
- Drag-and-drop interactions
- Processing indicators and animations
- Accessibility-friendly controls
- Keyboard shortcuts
- Smooth micro-interactions
- Client-side processing for improved privacy
