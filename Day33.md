#  Media Integrity Analyzer

An interactive single-file HTML application that teaches **media literacy** through guided, hands-on activities. Rather than testing prior knowledge, the simulator helps users discover how headlines, emotional language, and persuasive techniques influence the way information is consumed.

Designed with beginners in mind, the application combines interactive challenges with real-time feedback and educational explanations.

---

##  Overview

Media Integrity Analyzer is a browser-based educational simulator that helps users:

- Evaluate misleading headlines
- Detect emotional manipulation
- Understand media bias
- Improve critical thinking skills
- Learn practical media literacy habits

Everything runs completely offline in a single HTML file.

Learn how misleading headlines work.

After completing the simulator, users will understand:

- How headlines influence perception
- Emotional manipulation techniques
- Audience targeting strategies
- Media bias awareness
- Better critical thinking habits

##  Key Learnings

- Headlines don't always tell the full story.
- Emotional language can influence decision-making.
- Verifying information before sharing is essential.
- Critical thinking is one of the strongest defenses against misinformation.
