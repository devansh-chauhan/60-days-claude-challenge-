## Key Differences

| Aspect         | Lazy Prompt | Engineered Prompt |
| -------------- | ----------- | ----------------- |
| Clarity        | Basic       | Detailed          |
| Structure      | Simple      | Organized         |
| Examples       | Few or None | Multiple Examples |
| Visual Design  | Basic       | Professional      |
| Learning Value | Moderate    | High              |
| Context        | Limited     | Rich Context      |

---

## Key Learnings

1. Better prompts produce better outputs.
2. Context improves AI understanding.
3. Structured instructions lead to more accurate results.
4. Prompt engineering is not about making prompts longer; it is about making them clearer.
5. Well-designed prompts save time and improve output quality.

---

## Personal Reflection

This exercise showed me how important prompt engineering is when working with AI. A simple prompt can generate a usable result, but an engineered prompt produces more detailed, professional, and valuable outputs. Going forward, I will focus on providing clearer instructions, context, and desired outcomes when interacting with AI tools.
