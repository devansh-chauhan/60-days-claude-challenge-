#  Prompt Puzzle — Master AI Prompting Through Play

An interactive, browser-based learning game designed to help users improve their prompt engineering skills through hands-on challenges. Built as a single-file HTML application, Prompt Puzzle teaches users how to write better AI prompts by solving engaging puzzles instead of reading theory.

##  Overview

Prompt Puzzle transforms prompt engineering into a gamified experience where players build, optimize, and evaluate AI prompts across different domains and difficulty levels.

For this implementation:
- **Domain:** Software Development & Coding
- **Difficulty:** Intermediate

The application works completely offline and requires no installation or backend services.

---

##  Features

-  Domain and difficulty selection
-  Three interactive prompt engineering challenges
-  Randomized coding scenarios
-  Drag-and-drop prompt construction
-  Prompt optimization exercises
-  Best prompt selection challenge
-  Live scoring system
-  Time and move tracking
-  Hint system
-  Prompt DNA visualization
-  Prompt Performance Report
-  Replay with new randomized scenarios
-  Premium responsive dark UI

---

##  Challenge Types

### 1. Build the Prompt
Construct an effective AI prompt by arranging prompt blocks in the correct order.

### 2. Clean the Prompt
Improve poorly written prompts by removing unnecessary instructions and adding missing context.

### 3. Choose the Best Prompt
Compare multiple prompts and identify the one most likely to produce high-quality AI responses.

---

##  Performance Metrics

The simulator evaluates users based on:

- Accuracy
- Completion Time
- Number of Moves
- Wrong Placements
- Hints Used
- Optimization Bonus
- Overall Prompt Score

---

##  Final Performance Report

At the end of each session, users receive:

- Prompt Score
- Performance Rating
- Prompt Rank
- Prompt DNA Visualization
- Personalized Feedback
- Next Learning Milestone
- Optimized Final Prompt

---


##  Learning Outcomes

By completing this simulator, users will learn:

- How AI interprets prompts
- Prompt structuring techniques
- Importance of context and constraints
- Best practices for prompt optimization
- Differences between weak and effective prompts
- Practical prompt engineering principles

---

##  Key Learnings

- Better prompts lead to significantly better AI outputs.
- Clear context improves response quality.
- Specific constraints reduce ambiguity.
- Iterative prompt refinement is essential.
- Prompt engineering is a practical skill that improves with consistent practice.
