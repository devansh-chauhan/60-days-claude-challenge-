# Day 45 – Decision Strategist Dashboard

##  Project Overview

Day 45 of my **60 Days AI Challenge** focused on building an **Interactive Decision Strategist Dashboard** using a single HTML file.

The application simulates an AI-powered decision coach that helps users think more clearly about difficult choices through a structured interview, objective analysis, decision matrix, assumption testing, risk assessment, and a practical action plan.

Rather than making decisions for the user, the application encourages thoughtful reasoning and informed decision-making.

---

##  Objectives

- Practice structured decision-making
- Analyze trade-offs objectively
- Identify cognitive biases
- Evaluate multiple options using scoring
- Reduce uncertainty with practical action plans

---

##  Features

- Interactive four-question decision interview
- Decision summary
- Option-by-option analysis
- Assumption & bias identification
- Animated decision matrix
- Premortem risk analysis
- 7-Day decision validation plan
- AI-powered verdict
- Shareable decision summary cards
- Premium responsive dashboard
- Offline single-file application

---

##  Learning Outcomes

- Structured decision frameworks
- Critical thinking techniques
- Cognitive bias awareness
- Risk evaluation methods
- Dashboard UI design
- Interactive HTML, CSS & JavaScript development
